## 2. 思想・ビジョン・コンセプト

### 核となる3つの言葉

```
共育（きょういく）— 共に学び、共に育つ
共創（きょうそう）— 共に作り、共に生み出す
共助（きょうじょ）— 共に助け合い、支え合う
```

### 根本ビジョン

> AIと人が張り合うのではなく、それぞれの得意なことを活かしながら、
> 共に手を取り合って進んでいくことが、これからの未来にとって大切。
> AIの力によって、これまで何かを生み出したり、自分を表現したりすることに
> 苦手意識を持っていた人でも、少しずつ挑戦しやすくなる社会をつくる。

### 地域ビジョン

> 地方が誇れる新しい「あたり前」をみんなでつくっていく。
> 東京との格差を「埋める」のではなく、地方から価値を発信する。
> 仙台・東北から、共創の社会をつくる。

### 起業の原点（代表の問題意識）

- 今の日本には「余裕のなさ」が広がっている
- 「外注したいが想いをうまく伝えられない」という声を聞いた
- 「新技術を取り入れたいが何から始めればいいかわからない」という声
- AIという大きな波は止められない → だからこそ「人がAIと共に進む」道筋を示す

### 4ステップのストーリー（why）

### タグライン哲学

**「古き良き × 最先端」**

- 日本的な温かさ・人間らしさ（古き良き）を失わずに

### [D] AI搭載サポート（伴走型）

**一言説明：** AIを学ぶのではなく、あなた自身にAIを搭載する。自走できるまで伴走。

**コンセプトの核心**

```
「学ぶ」≠ 「アップデート」

学ぶ → 理解して、覚えて、知識を蓄える。全部わかってからでないと使えない。
アップデート → 更新を受け入れるだけで進化する。スマホと同じ。
```

**スマホ比喩（コンテンツ素材として重要）**

> スマートフォンのアップデート通知が来たとき、更新内容を全部理解してからボタンを押す人はほとんどいない。
> でもアップデート後は前より便利に、安全に、スムーズに使える。AIも同じ。

**一般的なAI講座との違い**

| 項目 | 一般的なAI講座 | AI搭載サポート |
|------|--------------|----------------|
| 目的 | 知識を学ぶ | 自分をアップデートする |
| スタイル | ツール理解が中心 | 実務に組み込む |
| 進め方 | 講義を受ける | AIと共に進む |
| 成果 | 学んで終わることも | 仕事・事業が前に進む |
| ゴール | AIを理解する | 自走できる状態になる |

**こんな方に向いている**
- AIに興味はあるが何から始めればいいかわからない
- 仕事をもっと早く進めたい
- 小さく始めて大きく変えたい
- ひとりで抱え込む働き方を変えたい
- AIと共に生きる感覚を持ちたい

**向いていない方（明示することで信頼を高めている）**
- 丸投げしたいだけの方
- 自分で試す気がない方
- 変わる意思がない方
- 完璧に理解しないと動けないと思っている方

**FAQ（よくある質問）**

| 質問 | 回答 |
|------|------|
| AI初心者でも大丈夫？ | まったく問題なし。音声入力や簡単な対話から始める |
| タイピングが苦手でも？ | 音声入力を活用するので問題なし |
| どんな業種でも相談できる？ | 個人事業主・副業・小規模事業者であれば業種問わずOK |
| ツールやアプリ制作の相談も？ | 業務効率化ツール・簡単なアプリの実装支援も可能 |
| 本当に学ばなくても大丈夫？ | 「体験しながら更新されていく」イメージ |
